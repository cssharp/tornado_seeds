#tornado框架MVC用例
1. 结构
``` 
models/ ------------------------模型，数据库映射
static/ ------------------------静态文件
template/-----------------------模板，html文件
upfile/-------------------------文件上传路径
views/--------------------------试图，业务逻辑
urls.py-------------------------路由
app.py--------------------------项目启动文件
config.py-----------------------配置文件
connect.py----------------------数据库连接
readme.md-----------------------README
requirements.txt----------------依赖包
```

2. 安装依赖
``` 
pip install -r requirements.txt
```

3. 启动项目
``` 
python app.py
```
